get the SPI control byte

Returns an SPI control byte.

The MCP23S17 is a slave SPI device. The slave address contains four fixed bits (0b0100) and three user-defined hardware address bits (if enabled via IOCON.HAEN; pins A2, A1 and A0) with the read/write command bit filling out the rest of the control byte:: 

+--------------------+
|0|1|0|0|A2|A1|A0|R/W|
+--------------------+
|fixed  |hw_addr |R/W|
+--------------------+
|7|6|5|4|3 |2 |1 | 0 |
+--------------------+
